# Web (Next.js on Vercel)

## Env
```
NEXT_PUBLIC_SUPABASE_URL=
NEXT_PUBLIC_SUPABASE_ANON_KEY=
SUPABASE_SERVICE_ROLE_KEY=
NEXT_PUBLIC_SITE_URL=
```

## Scripts
- npm run dev
- npm run build
- npm run start

## Deploy on Vercel
- Root directory: `web`
- Build command: `npm run build`
- Output directory: `.next`
- Install command: `npm i`
